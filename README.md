# TopOne Salon — Admin App

Internal admin app for TopOne Salon Richmond, built with Expo + React Native.

## Features

- **Home** — Manage banners (image, title, description, sort order)
- **Services** — Add, edit and delete salon services
- **Collections** — Manage yearly model photo collections
- **Messages** — Real-time customer messaging with booking request cards
- **Profile** — Update profile details and sign out
- **Locations** — Manage salon location info (address, postcode, phone, hours, etc.)

## Stack

- Expo Router (file-based routing)
- React Native + NativeWind (Tailwind)
- Supabase (auth, database, storage, realtime)

## Getting started

```bash
npm install
npx expo start --clear
```
